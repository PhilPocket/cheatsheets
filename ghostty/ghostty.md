<! MacBook 16" config update -->

## CLI

`$ ghostty`

*   `--help` - show help
*   `--version` - show version
*   `+list-actions` - list actions for keybindings
    *   `--docs` - show documentation too
*   `+list-fonts` - list fonts for configuration
*   `+list-keybinds` - list keybindings
*   `+list-themes` - list themes for configuration
*   `+show-config` - show user configuration (diff)
    *   `--default` - show default configuration
    *   `--docs` - show documentation too
*   `+validate-config` - validate configuration

## Keybindings

* `CMD + SHIFT + ,` - reload config

### Splits

*   create

    *   split down - `CMD + SHIFT + d`
    *   split right - `CMD + D`

*   go to

    *   split down - `CMD + OPT + DOWN`
    *   split left - `CMD + OPT + LEFT`
    *   split right - `CMD + OPT + RIGHT`
    *   split up - `CMD + OPT + UP`

*   resize

    *   split down - `CMD + CTRL + DOWN`
    *   split left - `CMD + CTRL + LEFT`
    *   split right - `CMD + CTRL + RIGHT`
    *   split up - `CMD + CTRL + UP`

*   close split - `CMD + W`

### Tabs

*   create tab - `CMD + T`
*   go to

    *   first tab - `CMD + 1`
    *   last tab - `CMD + 9`
    *   next tab - `CTRL + TAB`
    *   previous tab - `CTRL + SHIFT + TAB`
    *   second tab - `CMD + 2`

*   close tab - `CMD + OPT + W`

### Windows

*   create window - `CMD + N`
*   close
    *   window - `CMD + SHIFT + W`
    *   windows - `CMD + Q`
